# Print

Model: Custom, built from a Harper's logo in svg and OpenSCAD.

Slicer: Simplify3d

Filament: White PLA from Flashforge

Settings:
- Medium resolution
- Mostly default for Flashforge Creator Pro, using PLA, except:
    - Include raft
    - Extruder heat at 200C
    - Bed at 60C
